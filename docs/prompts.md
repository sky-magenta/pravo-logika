# Шаблоны запросов

Готовые формулировки для каждого режима. Работают и как слэш-команды (плагин Claude Code), и как обычный запрос (скилл подгрузится сам по `description`). Текст в `‹…›` замените своим.

## Ревью — разбор аргументации (диагностика)

```
/pravo-logika:review ‹правовая позиция или меморандум›
```
Обычным запросом:
```
Проверь правовую логику: ‹текст›
Разбери меморандум и назови дефекты аргументации: ‹текст›
Насколько логична эта позиция: ‹текст›
```
Даёт: вердикт, структуру позиции, таблицу дефектов права (рус. + лат.), пробелы в фактах, скрытые посылки, флаги к проверке. Текст не переписывает.

## Правка — исправить аргументацию

```
/pravo-logika:review ‹текст›   →   затем: «поправь»
```
Обычным запросом:
```
Поправь квалификацию, сохранив стиль: ‹текст›
Сделай позицию логичной, минимально меняя текст: ‹текст›
```
Даёт: диагностику + переписанный текст (минимальный дифф, неудержимые выводы ослаблены, флаги в тексте) + список правок.

## Clause-check — однозначность формулировки

```
/pravo-logika:clause-check ‹пункт договора›
```
Обычным запросом:
```
Проверь формулировку на двусмысленность: ‹пункт›
Однозначно ли это условие: ‹пункт›
```
Ловит: плывущие определения, перечни, союзы «и/или», пороги, кванторы, модальности, а также сроки/условия и механику закрытия (отлагательные условия, порядок исполнения, signing/closing, survival).

## Сравнение — карта спора двух позиций

```
/pravo-logika:compare
Позиция A: ‹довод первой стороны›
Позиция B: ‹довод второй стороны›
```
Обычным запросом:
```
Сравни две позиции — где стороны расходятся и что решает исход:
Истец: ‹…›
Ответчик: ‹…›
```
Даёт: предмет спора (об одном ли спорят — или мнимый спор), карту расхождений (право/факт/оценка), логическое соотношение позиций (противоречат / противны), скрытые посылки, несущую точку, дефекты сопоставления, флаги к проверке. Правоту не присуждает.

## Разбор нормы — текст нормы как объект

```
/pravo-logika:norm ‹текст статьи / пункта / положения устава›
```
Обычным запросом:
```
Разбери норму на элементы и найди неоднозначности: ‹текст нормы›
Разложи статью: гипотеза, диспозиция, санкция: ‹текст›
```
Даёт: структуру (гипотеза/диспозиция/санкция, модальность), кванторы и адресата, исключения, таблицу неоднозначностей в самой норме, внешние зависимости (отсылки).

---

**Во всех режимах:** скилл проверяет **форму** рассуждения, а не **каково** право. Нормы и практику по памяти не подставляет — ставит флаг `[требует проверки: норма / практика]`. Факт принимает как данность автора. Каталог дефектов — в [`cheatsheet.md`](cheatsheet.md), пример разбора — в [`example.md`](example.md).
